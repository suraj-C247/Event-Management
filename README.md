# Event-Management
Event Management Task


We have an event management system. The system has 15k+ Users.

Requirement:

Admin/ User Login, signup, change password, logout of basic details and user listing with active and deactivated

Admin/Users can create the event, once the event is created and saved in the database, the system should send an email invite notification to all the users - 

CONDITION - A notification email should get triggered after 1 hour of the event creation and users can able to accept or reject the invite.

User who has not accepted or rejected the invite, the system should send another email notification.
CONDITION - This notification should be sent 1 hour before the event only to the users who have not taken any action (Accept / Reject)


Event Management field 
- Event Title 
- Event Date & time 
- Event Description with text editor 
- Event Image with cropper 
- Event venue
